# 認領考古

在每次期中或期末考結束時，我們會建立一個 Google 試算表，並公布每個人應該要負責幾份考古，請大家能在考**完試完的隔天前**到 Google 表單上認領你想做的考古範圍，並在**負責人**的欄位上填上你的姓名。

原則上，所有的考古都必須在**上課隔天 17:00 前**完成，試算表的欄位中也會明列日期，大家可以對照著看。此外，在製作考古時也須注意到**年度**的欄位，詳細的用途將會在後面的教學中做說明。
